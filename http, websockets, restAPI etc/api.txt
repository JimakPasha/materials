Server API 
application programming interface

Это интерфейс взаимодействия с приложением
У любого сервиса есть эндпоинты
Эндпоинты содержат характеристики: По сути это и есть api
1 урлы на которые мы можем делать запросы
2 тип запроса (get, post ...)
3 request payload (что с запроса слать)
4 response data (ответ)
5 ответы коды 200, 300, 400, 500
